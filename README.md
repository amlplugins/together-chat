# @amlplugins/together-chat

Native `together-ai` SDK re-exported under the `@amlplugins` namespace.

> Together AI Chat — 200+ open-source LLMs (Llama, Mixtral, DeepSeek, Qwen, Gemma) with streaming and tool use.

## Install

```bash
npm install @amlplugins/together-chat
```

## Use

```ts
import { /* named exports from together-ai */ } from "@amlplugins/together-chat";
// or
import sdk from "@amlplugins/together-chat";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
